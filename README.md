# elder_companion_app
The Elder Companion App is designed to support elderly individuals in their daily lives. and help
them stay safe,healthy,and connected with their families.

## Features:
- Medicine reminders
- Doctor appointment reminders
- Daily exercise guidance
- Emergency calling
- Tracking
- SMS alert to family
- Video calling with family
- Voice calling with family
- Location sharing for safety
- Health tips for elderly wellness
- Voice assistant for easy navigation

## Technologies Used:
- python
- flask
- HTML
- CSS
- JavaScript (Voice Assistant)

This app helps senior citizens stay safe, healthy, and connected with their families.
